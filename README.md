# Guess-My-Number
Simple game where you have to guess a random number between 1 - 20. 
This game keeps track of score and also updates a highscore value if you beat your personal best. 
The winning number will be hidden until you have guess the winning number correctly. 
To make the game a bit easier to play, hints are also given to allow player to know if the number they have guessed was either too high or too low. 

Future planned features
- create multiple ranges of winning numbers for the player to choose from
  i.e : 1 - 20, 1 - 40, 1 - 100, etc.
